# AI Assignment Tutor

A premium, AI-powered platform for students and teachers to manage and analyze assignments.

## 🚀 How to Run the Project

The project is built with **React** and **Vite**. Follow these steps to start the development server:

1.  **Open a Terminal** in the project root directory .
2.  **Install Dependencies** (if not already done):
    ```bash
    npm install
    ```
3.  **Start the Dev Server**:
    ```bash
    npm run dev
    ```
4.  **Access the App**:
    Open your browser and navigate to:
    [http://localhost:5173/](http://localhost:5173/)

## 🛠 Features

- **Authentication**: Register as a Student or Teacher.
- **Student Portal**: Upload PDF, Docx, or Images; get AI hints and full solutions.
- **Teacher Portal**: Review student-uploaded work, run AI content analysis, and assign grades.
- **Multi-language**: Seamlessly switch between English, Hindi, and Spanish.

## 📁 Project Structure

- `src/App.jsx`: Main application logic and state management.
- `src/components/Auth.jsx`: Authentication UI.
- `src/components/StudentDashboard.jsx`: Student-facing tools.
- `src/components/TeacherDashboard.jsx`: Teacher-facing analytics and grading.
- `src/index.css`: Glassmorphism design system.
